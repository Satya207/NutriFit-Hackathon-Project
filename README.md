# NutriFit – Health & Fitness Hackathon Project

## Overview
NutriFit is a health and fitness-focused web application developed during the 24-hour VEDIC VISION 2K25 Hackathon held on August 14–15, 2025.

The project was created to encourage users to maintain a healthy, active, and consistent lifestyle in today’s technology-driven world.

Our team “Tech Troupe” received a Special Prize at the hackathon for the project.

## Features
- Workout guidance
- Healthy lifestyle support
- Daily tasks & consistency tracking
- User activity monitoring & management

## Technologies Used
- HTML
- CSS
- JavaScript
